# dotenv-linter
https://pypi.org/project/autopub/

## Installation

### install rust
https://www.rust-lang.org/tools/install

### install dotenv-linter
```
cargo install dotenv-linter
```

## Usage

### Lint files
```
dotenv-linter
```

### auto fix lint errors
```
dotenv-linter fix
```

### compare .env.example and .env
```
dotenv-linter compare .env.example .env.dev .env.staging .env.production
```
